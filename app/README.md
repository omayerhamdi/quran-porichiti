# আল-কুরআন: এক নজরে

কুরআনুল কারীমের পরিচিতিমূলক ইন্টারেক্টিভ ওয়েব অ্যাপ্লিকেশন — বাংলা ও আরবি, ৬টি পাতা।
কনটেন্টের উৎস: `../quran-porichiti-content.md` (মাস্টার ডকুমেন্ট)।

## চালানো

কোনো বিল্ড-স্টেপ নেই — pure HTML/CSS/JS। যে কোনো স্ট্যাটিক সার্ভারে চলবে:

```bash
python3 -m http.server 8517 --directory app
# তারপর ব্রাউজারে: http://localhost:8517
```

ডাবল-ক্লিকে `index.html` খুললেও চলবে (সব অ্যাসেট লোকাল ও রিলেটিভ পাথে)।

## ডিপ্লয়

`app/` ফোল্ডারটি যেমন আছে তেমনই GitHub Pages / Netlify / Vercel / যেকোনো
স্ট্যাটিক হোস্টে আপলোড করলেই হবে। কোনো সার্ভার-সাইড নির্ভরতা নেই; ফন্টসহ
সবকিছু লোকাল, তাই অফলাইনেও কাজ করে।

## গঠন

| ফাইল | কাজ |
|---|---|
| `index.html` | শেল — হেডার, নেভিগেশন, ফুটার |
| `js/data.js` | **সমস্ত কনটেন্ট** — টেক্সট বদলাতে শুধু এ ফাইল সম্পাদনা করুন |
| `js/app.js` | রাউটার, রেন্ডারার, কুইজ, অ্যানিমেশন |
| `css/style.css` | ডিজাইন সিস্টেম (লাইট + ডার্ক থিম) |
| `fonts/` | Noto Serif Bengali, Hind Siliguri (বাংলা); Amiri (আরবি) — লোকাল |
